### 简介
- 一个常见的 filter 多选逻辑，本质上是另一个 checkbox-group
- 添加 `items/template` 接口
- 统一所有的 `onChagne` 事件

### 预览

> 显示日期和时间。

```jsx live
<AcRangePicker
  onChange={(e) => {
    console.log('range evt: ', e.target.value);
  }}
/>
```

> 仅显示日期

```jsx live
<AcRangePicker style={{ width: '100%'}} showTime={false} format="YYYY-MM-DD" />
```
