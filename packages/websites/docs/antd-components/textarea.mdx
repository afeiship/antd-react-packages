### 简介
- 一个常见的 filter 多选逻辑，本质上是另一个 checkbox-group
- 添加 `items/template` 接口
- 统一所有的 `onChagne` 事件

### 预览
```jsx live
<AcTextarea
  defaultValue="abc"
  autoSize={{ minRows: 2, maxRows: 10 }}
  placeholder="Please input text..."
  onChange={(e) => {
    console.log('evt: ', e.target.value);
  }}
/>
```
