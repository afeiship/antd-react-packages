### 简介
- 一个常见的 filter 多选逻辑，本质上是另一个 checkbox-group
- 添加 `items/template` 接口
- 统一所有的 `onChagne` 事件

### 预览
```jsx live
<Space direction="vertical">
    <blockquote>
        <h4>Default style</h4>
        <AcRadioGroup items={data.items1} />
    </blockquote>

    <blockquote>
        <h4>Default style with Raw template</h4>
        <AcRadioGroup items={['opt1','opt2','opt3']} template={tpls.radioRaw} />
    </blockquote>

    <blockquote>
        <h4>Button style</h4>
        <AcRadioGroup
          items={data.items1}
          template={tpls.radioKv}
          templateOptions={{ isButton: true }}
          buttonStyle={'solid'}
        />
    </blockquote>
</Space>
```
