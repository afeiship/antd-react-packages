### installation
- 直接用 `npm` 或者 `yarn` 安装。
- 统一 `event` 为 `e.target.value` 取值

```bash npm2yarn
npm install @jswork/antd-components
```

### usage
```js
import { AcInputToken } from '@jswork/antd-components';
```
