### installation
- 直接用 `npm` 或者 `yarn` 安装。
- 添加 `items/template` 通用属性
- 统一 `event` 为 `e.target.value` 取值

```bash npm2yarn
npm install @jswork/antd-components
```

### usage
```js
import { AcCheckboxGroup } from '@jswork/antd-components';
```

### preview
```jsx live
<Space direction="vertical">
    <blockquote>
        <h4>默认 `template/value-label`</h4>
        <AcCheckboxGroup
            value={['a']}
            onChange={e=>console.log('evt1: ',e.target.value)}
            items={[
                { value: 'a', label: 'A' },

    <blockquote>

            items={['a','aaa','abc','xyz']}

        />
    </blockquote>
</Space>
```
