### installation
- 直接用 `npm` 或者 `yarn` 安装。
- 添加 `items/template` 通用属性
- 统一 `event` 为 `e.target.value` 取值

```bash npm2yarn
npm install @jswork/antd-components
```

### usage
```js
import { AcCheckboxGroup } from '@jswork/antd-components';
```

### preview
```jsx live
<AcCheckboxGroup
    items={[
        { value: 'a', label: 'A' },
        { value: 'b', label: 'B' },
        { value: 'c', label: 'C' },
        { value: 'd', label: 'D' }
    ]}
/>
```
