### 简介
- 一个常见的 filter 多选逻辑，本质上是另一个 checkbox-group
- 添加 `items/template` 接口
- 统一所有的 `onChagne` 事件

### 预览
```jsx live
<Space>
    <AcCheckbox value={true} onChange={(e) => { console.log('evt1 : ', e.target.value) }}>
      I Agree
    </AcCheckbox>
    <AcCheckbox onChange={(e) => { console.log('evt2 : ', e.target.value) }}>
      I Don't Agree
    </AcCheckbox>
</Space>
```
