### 简介
- 一个常见的 filter 多选逻辑，本质上是另一个 checkbox-group
- 添加 `items/template` 接口
- 统一所有的 `onChagne` 事件

### 预览
```jsx live
<AcTreeSelect
  style={{ width: '100%' }}
  defaultValue={['0-0-1-0']}
  items={data.items4}
  onChange={(e) => {
    console.log('e: ', e.target.value);
  }}
/>
```
