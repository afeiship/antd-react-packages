### 简介
- 一个常见的 `filter` 多选逻辑，本质上是另一个 `checkbox-group`
- 添加 `items/template` 接口
- 统一所有的 `onChagne` 事件
- 增加 `disabled` 属性
- 增加 `toggleable` 属性
- 增加 `closeable` 属性，右侧显示小的 `close icon`

### 预览
```jsx live
<Space direction="vertical">
  <ul>
    <li><AcCheckableTag value={true} closeable>显示右边 close icon</AcCheckableTag></li>
    <li><AcCheckableTag closeable toggleable>显示close icon，并且点击可切换 close 的有与无</AcCheckableTag></li>
    <li><AcCheckableTag disabled> Disabled 状态 </AcCheckableTag></li>
    <li>
        <AcCheckableTag onChange={e=>console.log('evt: ', e.target.value)}>
        可以有 onChange 事件
        </AcCheckableTag>
    </li>
</ul>
</Space>
```
