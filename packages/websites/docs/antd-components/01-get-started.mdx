### 简介
> 主要是提供了一套 `antd` 可交互组件(主要是Form组件)的二次封装，让其接口更加的统一，更加的简单，更加的方便。

### 安装
- 直接用 `npm` 或者 `yarn` 安装。
- 目前兼容 `antd4/antd5`，但长期维护的是 `antd5`

```bash npm2yarn
npm install @jswork/antd-components antd
```

## 使用
```js

import {
  AcSelect,
  AcCheckbox,
  AcCheckboxGroup,
  AcInput,
  AcInputNumber,
  AcRadioGroup,
  AcRate,

  AcDatePicker,
  AcTextarea,
  AcTransfer,
  AcRangePicker,
  AcSearch,
  AcUploadDragger,
  AcCheckableTag,
  AcCheckableAll,
  AcPreSelect,
  AcBreadcrumb,
  AcEditableTagGroup,
  AcTree,
  AcTreeSelect,
  AcUploadPictureCard,
  AcTimePicker,
  AcSlider,
  AcSliderRange

// 引入 SCSS
```
