### 简介
> 主要是提供了一套基于 `antd` 实现的 `form-schema`，用于方便的生成表单，支持 `antd` 的所有组件，同时支持自定义组件。

### 安装
- 直接用 `npm` 或者 `yarn` 安装。
- 目前兼容 `antd4/antd5`，但长期维护的是 `antd5`
- 并非完全自主开发，基于 `rekit/antd-form-builder` 二次开发

```bash npm2yarn
npm install @jswork/antd-form-builder
```

### 使用
```js
// 引入JS
import FormBuilder from '@jswork/antd-form-builder';
```
