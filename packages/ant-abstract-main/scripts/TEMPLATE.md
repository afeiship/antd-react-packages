## installation
```shell
npm install -S @jswork/ant-abstract-main
```

## usage
1. import css
  ```scss
  @import "~@jswork/ant-abstract-main/dist/style.css";

  // or use sass
  @import "~@jswork/ant-abstract-main/dist/style.scss";

  // customize your styles:
  $ant-abstract-main-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
